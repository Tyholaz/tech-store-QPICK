## 🌟 Features

- 🛍️ Product catalog with category filtering
- 🔍 Product detail pages
- 🛒 Shopping cart and wishlist
- ✅ Checkout and order confirmation flow
- 📄 Contacts and terms pages
- 📱 Fully responsive design for all devices

## 🛠️ Built With

- HTML5
- CSS3 / Tailwind CSS
- Vanilla JavaScript

## 📂 Getting Started

```bash
git clone https://github.com/Tyholaz/tech-store-QPICK.git
cd tech-store-QPICK
```
Then simply open `index.html` in your browser — no build step required.
