# 🖥️ QPICK - Tech Store Website

A fully functional multi-page tech store website with modern UI and complete shopping flow.

## 🌟 Features

- Product catalog and product detail page
- Shopping cart and wishlist
- Checkout and order confirmation flow
- Contacts and terms pages
- Responsive design for all devices
